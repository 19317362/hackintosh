# hackintosh

## 系统支持

macOS mojave 10.14Beta - 10.14.5(18F132)

## 平台简介

HP ZHAN99 WorkStation G1（惠普战99）

### 主要配置

* Intel Core I7-8750h 6C12T
* Crucial DDR4 2666MHz 8G*2
* UHD630 （Mojave目前仍不支持N卡）
* Realtek Alc295
* Broadcom BCM94352hmb （M.2/NGFF)
* Samsung pm961

### 完成功能

* CPU12阶睿频
* 内建视网膜显示器
* AppleALC驱动声卡
* Wi-Fi驱动&Airport 
* 蓝牙
* 原生电池控制
* USB3.1
* 有线网卡驱动
* 摄像头驱动
* GPS
* 开机不会无背光
* 亮度可调
* 触摸板
* 睡眠以及唤醒

### 未完成：

* 指纹无解
* 雷雳、读卡器没有驱动

### 总体描述：

**基本完美**
